# Technical Setup

## Unity Configuration

- Unity Version: 2021.3.1f1
- Target Platform: PC, Mac & Linux Standalone

### Project Settings:
- Graphics API: Direct3D11
- Player Settings: Set resolution, display settings, and other player options according to the project requirements.

## Technical Requirements

- Minimum System Requirements:
  - OS: Windows 10 64-bit
  - Processor: Intel Core i3 or equivalent
  - Memory: 4 GB RAM
  - Graphics: NVIDIA GeForce GTX 660 or equivalent
  - DirectX: Version 11
  - Storage: 10 GB available space

- Recommended System Requirements:
  - OS: Windows 10 64-bit
  - Processor: Intel Core i5 or equivalent
  - Memory: 8 GB RAM
  - Graphics: NVIDIA GeForce GTX 970 or equivalent
  - DirectX: Version 11
  - Storage: 10 GB available space

Make sure to install all required packages and set up the environment before running the application.
