# Spaceship Horror Demo

## Project Overview
This repository contains a demo project for a spaceship-themed horror game. The objective is to create a suspenseful and immersive experience for players while navigating through an abandoned spaceship filled with strange occurrences and terrifying entities.

## Features
- **Exploration:** Players can explore different sections of the spaceship.
- **Puzzles:** Mechanics that require players to solve puzzles to progress.
- **Horror Elements:** Atmospheric sound design, lighting, and unique enemy encounters.
- **Visuals:** High-quality graphics and detailed environments.

## Project Structure
```
spaceship-horror-demo/
├── assets/            # Contains all game assets (images, sounds, etc.)
├── src/              # Source code for the game
│   ├── main.py       # Main game file
│   ├── player.py     # Player class and methods
│   └── enemy.py      # Enemy behavior and interactions
├── README.md         # Project overview and documentation
└── requirements.txt   # Python dependencies
```

## Installation
1. Clone this repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd spaceship-horror-demo`
3. Install dependencies: `pip install -r requirements.txt`

## Usage
Run the game using the command: `python src/main.py`
